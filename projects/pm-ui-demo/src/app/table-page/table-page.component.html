Table

The pm-table component can be used in two ways: data driven, or content driven. A driven table is best used when you have an array of objects that are in a predictable structure.

For example, an array of users where each user object has a "firstName", "lastName" and "address" property. You start by telling the table what your columns are. You do this by passing an array of TableColumn objects into the [columns] property. You pass the table's data in via the [rows] property.

While using a data driven table, you can also use the pm-paginator component. Control the page size options via the [pageSizes] property like this:
<pm-paginator [table]="tableTemplateRef" [pageSizes]=[10,20,30]></pm-paginator>
The default page sizes are 5, 10 and 15.

Data driven table example:

    <table
    #exampleTable="pm-table"
    pm-table
    [columns]="[
      { key: 'firstName', label: 'First Name' },
      { key: 'lastName', label: 'Second Name' },
      { key: 'address', label: 'Address' }
    ]"
    [rows]="[
      { firstName: 'Mario', lastName: 'Mario', address: '101 Star Kingdom Rd' },
      { firstName: 'Luigi', lastName: 'Mario', address: '103 Star Kingdom Rd' },
      {
        firstName: 'Princess',
        lastName: 'Peach',
        address: '325 Royal Mushroom Ln'
      },
      { firstName: 'Bowser', lastName: 'Koopa', address: '478 Lava Lake Rd' },
      {
        firstName: 'Madame',
        lastName: 'Flurry',
        address: '89 Boggly Woods Dr'
      },
      { firstName: 'Don', lastName: 'Pianta', address: '278 W Rogue Port St' },
      {
        firstName: 'Professor',
        lastName: 'Frankly',
        address: '2765 E Rogue Port St'
      },
      { firstName: 'Prince', lastName: 'Mush', address: '909 Gladiator Blvd' }
    ]"
    ></table>
    <pm-paginator [table]="exampleTable"></pm-paginator>


A content driven table is just another way of saying a plain ol' fashion html table. All you need for that is to add the "pm-table" selector to a table element like so:
<table pm-table>...</table>
Note that you can not use the "pm-paginator" component while using a content driven table.

Content Driven Table Example:

    <thead>
      <tr>
        <th>First Name</th>
        <th>Last Name</th>
        <th>Address</th>
      </tr>
    </thead>
