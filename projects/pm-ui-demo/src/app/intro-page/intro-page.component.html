Welcome to the demo site for pm-ui , the Paper Mario inspired UI library! This UI library was created due to my love for the game, and I do not claim any rights/ownership of the Paper Mario franchise. All rights go to Nintendo. This is just a little side project that I am using to learn different angular concepts, explore UI/UX designs/patterns such as brutalism as well as accessability. Feel free to use this library however you want. This site offers documentation and examples for each component in the library. The site itself is also built using these components.
