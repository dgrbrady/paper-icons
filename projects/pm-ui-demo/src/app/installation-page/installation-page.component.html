Installation

  • In your angular workspace, run npm i @drbrady/pm-ui
  • In your app.component.module.ts you can import specific modules if you only need one component, or you can import all PmUi components by importing the PmUiModule
  • Finally, in your angular.json file, add this file path to the "styles" array: "./node_modules/@dgrbrady/pm-ui/src/lib/styles.css" Thank-a you for checking it out! You're a star!
