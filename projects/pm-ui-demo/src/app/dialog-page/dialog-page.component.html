Dialog

The <pm-dialog></pm-dialog> component can be used to add a scrolling effect where the letters are rendered one at a time.

Features:
  • Click the dialog to skip the animation.
  • Comes with 3 predetermined scrolling speeds, slow , normal , and fast . You can also provide an arbitrary number if the speeds don't work for you.
  • Ability to set the tone of the dialog. The only one currently implemented is scared
Examples:
  • <pm-dialog speed="slow"><pm-dialog> This is a dialog using the "slow" speed
  • <pm-dialog><pm-dialog> This is a dialog using the "normal" speed.
  • <pm-dialog speed="fast"><pm-dialog> This is a dialog using the "fast" speed
  • <pm-dialog [speed]="100"><pm-dialog> This is a dialog using a custom speed. The number is how many milliseconds should be delayed between each character.
  • <pm-dialog tone="scared"><pm-dialog> Shaking in your boots??? Use the "scared" tone! Boooooooooo!!
