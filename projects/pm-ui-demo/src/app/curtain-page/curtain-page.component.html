Curtains

The pm-curtains component is a Paper Mario flavored tab component. To create a new tab, use the pm-curtain-tab component. Setting the "label" attribute on the pm-curtain-tab component will set the label for that tab. The tab content goes in between the opening and closing tags.

pm-curtains with default colors: 
<pm-curtains>
    <pm-curtain-tab label="first">
        first tab
    </pm-curtain-tab>
    <pm-curtain-tab label="second">
        second tab
    </pm-curtain-tab>
    <pm-curtain-tab label="third">
        third tab
    </pm-curtain-tab>
</pm-curtains>
